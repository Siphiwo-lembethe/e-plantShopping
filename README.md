#eClone Shopping

eClone Shopping is a simple online store bulit with React.
It let's users browse products, add to cart, and see totals - all in one place.

##How to Run
1. Clone the repo
   'git clone https://github.com/Siphiwo-lembethe/e-plantShopping.git
2. Open in VS Code or browser
3. Click "Get Started" => start shopping!

4. Built for IBM Coursera React course.
